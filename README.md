# chrome-lunchguiden-umea

Google Chrome extension / plugin giving you the daily lunch menus for restaurants around Ume√•. [This](https://github.com/yeoman/generator-chrome-extension) generator was used for generating the architecture.

## Setup
Nodejs need to be installed. Install it [here](https://nodejs.org/en/).
```bash
# Install gulp
npm install -g gulp-cli

# Install dependencies
npm install
bower install

# Run gulp watch
gulp watch
```
